Estructura básica de una página en HTML5

En HTML5, <nav> es un elemento que define una sección con enlaces de navegación, como un menú principal o una lista de enlaces a otras páginas del sitio web.

En HTML5, una sección (<section>) representa una sección genérica de un documento o aplicación, que generalmente se compone de un contenido temático con un encabezado.

En HTML5, la etiqueta <article> se utiliza para representar contenido independiente y autocontenido que puede ser distribuido o reutilizado de manera independiente

En HTML5, la etiqueta <aside> se utiliza para representar contenido que está relacionado con el contenido principal de una página, pero no es esencial para su comprensión

En HTML5, el elemento <footer> representa la sección de pie de página de una página web, un documento o una sección, generalmente ubicada en la parte inferior.
